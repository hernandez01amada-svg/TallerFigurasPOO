# Taller POO - Figuras Geométricas
 taller de Programación Orientada a Objetos (POO) sobre jerarquía de figuras geométricas en Python.
## Requisitos 
- **Encapsulamiento:** Implementado mediante el uso de `@property` y `@setter` con validaciones de datos (dimensión > 0).
- **Herencia:** Las clases `Cuadrado`, `Rectangulo` y `Circunferencia` heredan de la clase base `FiguraGeometrica`.
- **Polimorfismo:** Funciones como `sumar_areas()` y `sumar_perimetros()` en `main.py` operan correctamente sobre cualquier objeto de la jerarquía.
## Estructura de Archivos
- `figura_geometrica.py`: Clase abstracta base que define atributos y validaciones.
- `cuadrado.py`: Clase hija que calcula área y perímetro específicos.
- `rectangulo.py`: Clase hija que calcula área y perímetro específicos.
- `circunferencia.py`: Clase hija que utiliza la constante matemática PI.
- `main.py`: Script principal para la instanciación de figuras y demostración del polimorfismo.
